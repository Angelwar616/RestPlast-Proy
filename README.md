Automated Plastic Conversion Monitoring System
Overview

This project implements a full-stack monitoring and data management platform designed for an automated plastic conversion process. The system integrates web technologies, cloud services, and IoT communication to collect, store, and visualize process data in real time.

The platform allows users to monitor sensor variables, review historical process data, and supervise system performance through an interactive web interface.

System Architecture

The system follows a distributed architecture composed of several components:

IoT Data Source – Sensors and microcontrollers generate process data.

Cloud Communication Layer – Data is transmitted and synchronized through cloud services.

Backend API – Responsible for data processing, storage management, and communication with the frontend.

Frontend Application – Web interface used to visualize and manage the system data.

This architecture enables scalable data handling and modular system expansion.
